Bento's configuration is split across up to four files that resolve in a fixed priority order.

- [Overview](/configuration/overview) — Config file locations, the layering model, and built-in visibility tools.
- [Output](/configuration/output) — Container format, destination, filename templating, conflict resolution, and metadata.
- [Video](/configuration/video) — Encoder, CRF, preset, and source preprocessing.
- [Audio](/configuration/audio) — Codec, bitrate, mixdown, normalization, and per-track configuration.
- [Subtitles](/configuration/subtitles) — Soft mux, burn-in, external sidecars, ASS filtering, and track subtraction.
